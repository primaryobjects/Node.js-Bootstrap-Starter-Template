Node.js Bootstrap Starter Template

A quick and easy Node.js + Jade template project, using Twitter Bootstrap.

Demo:
http://nodebootstrapper.herokuapp.com

Modules required:
express
jade
stylus

Contents:

layout.jade
header.jade
footer.jade
index.jade

Quick Start:

Open a command prompt and type: npm install
Next, edit index.jade to add your content. The default contents include:

extends layout

block content
	div.container
		h1 Program Name
		p Hello World
